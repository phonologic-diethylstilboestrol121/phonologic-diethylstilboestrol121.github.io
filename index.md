# 🕹️ pkmanager - Manage your classic Pokémon save files

[![](https://img.shields.io/badge/Download-pkmanager-blue)](https://github.com/phonologic-diethylstilboestrol121/pkmanager)

## 📋 About this project

pkmanager allows you to manage save files from Pokémon games released between the Game Boy Advance and Nintendo 3DS eras. This includes Generations 3 through 7. You can upload your save files, edit your Pokémon, and view them through an integrated browser emulator. The application uses the core logic from PKHeX to ensure your data stays compatible with the original games. You can also organize your collection using a built-in storage system similar to the official Pokémon Bank.

## ⚙️ System Requirements

To run pkmanager on your Windows computer, confirm that your system meets these requirements:

- Operating System: Windows 10 or Windows 11.
- Memory: 4 GB of RAM or more.
- Browser: An updated version of Google Chrome, Microsoft Edge, or Firefox.
- Storage: 200 MB of free space for temporary files.
- Internet: A stable connection for initial setup and emulator resources.

## 🚀 Getting Started

Once you open the software, you will see a clean dashboard. Follow these steps to begin managing your collection:

1. Click the "Open Save File" button.
2. Browse your computer folders to select your game save file. Accepted formats include common battery files from your physical carts or emulator data.
3. Once the file loads, the interface displays your Pokémon box data.
4. Select a specific Pokémon to view individual stats and legality status.
5. Use the editing tools to change names, items, or held statuses. 
6. Click "Save Changes" to write the data back to your file.

## 🎮 Using the Integrated Emulator

The application includes an emulator that runs directly in your browser. This allows you to test your edits without moving files to a separate console or handheld device.

1. Navigate to the "Emulation" tab inside the software.
2. Select the game generation that matches your save file.
3. The software will download necessary assets. This only happens once.
4. Click the "Play" button to launch the browser window.
5. Provide your path to your game ROM when prompted.
6. Use your keyboard or a connected controller to play.

## 🛡️ Legality and Safety

The software includes a tool that checks your Pokémon against the official game rules for data structure and move sets. This is known as a legality analysis. 

- Green status: The Pokémon is valid and behaves like a creature caught naturally in the game.
- Yellow status: The Pokémon consists of data that might cause issues during local trades.
- Red status: The Pokémon contains illegal data that the game engine may reject or crash upon loading.

Always create a backup of your original save file before you make any edits. Copy your file to a safe folder on your hard drive to prevent data loss.

## 🗄️ Working with Pokémon Bank

The repository includes a digital storage feature. This functions as a local repository for your creatures. 

1. Go to the "Bank" tab.
2. Create a new storage folder.
3. Drag and drop Pokémon from your current save file into this space.
4. You can keep these creatures here even after you load a different save file.
5. Move them back to any compatible game whenever you choose.

## 🛠️ Troubleshooting

If you encounter issues, verify the following:

- File Corruption: If a save file fails to load, ensure it is not locked by an active emulator session. Close all other programs using that save file before you open it in pkmanager.
- Browser Caches: If the in-browser emulator fails to start, clear your internet history and cache. Refresh the page to trigger a fresh download of the emulation tools.
- Permissions: If the application cannot save your changes, check that your save file is not set to "Read-only" in the Windows file properties.
